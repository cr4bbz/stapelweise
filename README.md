# Stapelweise

Moderne Karteikarten-Lernplattform mit Spaced Repetition. Kostenlos lokal, günstig in der Cloud. Die Open-Source-Alternative zu Anki.

## Status

🚧 In früher Entwicklung — MVP in Arbeit.

## Vision

Eine Lern-App, die so gut aussieht und sich so intuitiv bedient wie moderne Software aus 2026 – mit der algorithmischen Power von Spaced Repetition unter der Haube.
